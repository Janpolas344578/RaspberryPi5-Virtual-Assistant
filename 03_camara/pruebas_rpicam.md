# Pruebas de Cámara

## Detectar cámaras

```bash
libcamera-hello --list-cameras
```

o

```bash
rpicam-hello --list-cameras
```

## Vista previa

```bash
rpicam-hello
```

## Capturar fotografía

```bash
rpicam-jpeg -o foto.jpg
```

## Capturar segunda fotografía

```bash
rpicam-jpeg -o foto2.jpg
```
