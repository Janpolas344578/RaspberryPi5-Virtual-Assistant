# Actualización del Sistema

## Objetivo

Actualizar los paquetes y repositorios de Raspberry Pi OS.

## Comandos

```bash
sudo apt update
sudo apt upgrade -y
```

## Resultado esperado

- Repositorios actualizados.
- Paquetes del sistema actualizados.
- Corrección de vulnerabilidades y errores.

## Notas

Se recomienda ejecutar este procedimiento después de una instalación nueva del sistema operativo.
