# Visión Artificial con Moondream

## Objetivo

Utilizar el modelo Moondream mediante Ollama para describir imágenes.

## Ejecución

```bash
python probar_vision.py
```

## Resultado

El modelo fue capaz de:

- Detectar personas.
- Describir vestimenta.
- Reconocer objetos.
- Analizar fotografías de mascotas.

### Ejemplo de salida

```text
The image shows a young man with long hair wearing glasses...
```

### Segundo ejemplo

```text
The image features a white dog wearing a red bandana...
```

## Observaciones

La inferencia se ejecutó localmente en la Raspberry Pi utilizando Ollama y el modelo Moondream.
